# SME-Operations-Management-System

Project Overview

This project focuses on analysing and improving operational processes within a Small and Medium Enterprise (SME) environment offering:

  - Food services-
  - Printing services
  - Pool table services

The project was developed as part of a Process Testing and Systems Analysis academic project aimed at identifying operational inefficiencies and proposing a structured digital solution to improve business processes, service tracking, and transaction management.

Business Problem

The SME relied heavily on manual processes including:

  - Paper-based food order tracking
  - Informal stock management
  - Untracked printing requests
  - No structured pool table tracking
  - Poor transaction recording

These challenges resulted in:

  - Lost or incorrect orders
  - Service delays
  - Customer disputes
  - Limited financial visibility
  - Operational inefficiencies
  - Proposed Solution

An integrated web-based management system was proposed to centralise SME operations into a single platform capable of:

  - Managing food orders
  - Tracking printing jobs
  - Monitoring pool table sessions
  - Recording transactions
  - Supporting reporting and operational monitoring
